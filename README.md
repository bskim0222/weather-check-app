# 웨더체크

기상청, Yr.no, Windy 예보와 사용자의 현장 제보를 함께 보고
"지금 여기 비 와?" 같은 질문에 바로 답해주는 모바일 날씨 판정 앱입니다.

현재 단계는 실제 앱 개발 전 목업과 기획을 정리하는 단계입니다.
최종 목표는 Android와 iOS 앱스토어 출시입니다.

## 현재 기준 파일

- `weather-compass-full-mockup.html`: 현재 메인 전체 목업
- `weather-decision-card-weather-variants.html`: 날씨별 판정카드 디자인 시안
- `assets/`: 기상청, Yr.no, Windy 아이콘 파일

## 지금 정한 방향

- 모바일 우선 앱
- 첫 화면은 현재 위치 기준 판정 화면
- 상단 검색창에서 자연어 질문 입력
- 판정 탭은 "그래서 지금 어떤 날씨인가"를 답함
- 지도 탭은 주변 현장 제보를 위치로 보여줌
- 제보 탭은 사용자가 현장 날씨를 쓰고, 다른 사람 요청에 답함
- 비교 탭은 기상청, Yr.no, Windy 예보를 시간별/날짜별로 비교함

## 개발 방향

Android와 iOS 출시를 목표로 하므로 웹앱만 만들기보다는
React Native + Expo 기반으로 실제 모바일 앱을 만들 계획입니다.

HTML 목업은 버리지 않고, 실제 앱 화면을 만드는 기준 설계도로 사용합니다.

## 다음 순서

1. 기획서와 출시 로드맵을 기준으로 MVP 범위를 확정
2. Expo 앱 프로젝트 생성
3. 목업 화면을 React Native 컴포넌트로 옮기기
4. 샘플 데이터로 화면 동작 완성
5. 날씨 API, 위치 권한, 제보 저장 기능 연결
6. 테스트 빌드 후 Android/iOS 스토어 제출 준비
