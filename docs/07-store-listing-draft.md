# Store Listing Draft

This is a first draft for App Store and Google Play listing material.

## App Name

웨더체크

## Subtitle

예보와 현장 제보를 함께 보는 날씨 판정

## Short Description

기상청, Yr.no, FMI ECMWF 예보와 주변 현장 제보를 비교해 지금 필요한 날씨 판단을 도와줍니다.

## Full Description Draft

웨더체크는 "잠실운동장 지금 비 와?"처럼 궁금한 장소와 시간을 질문하면 여러 예보와 주변 현장 제보를 함께 비교해 보여주는 날씨 판정 앱입니다.

주요 기능:

- 현재 위치 기준 날씨 판정
- 장소/시간을 담은 자연어 질문
- 기상청, Yr.no, FMI ECMWF 예보 비교
- 시간별/날짜별 예보 비교표
- 주변 사용자의 현장 날씨 제보
- 특정 장소에 대한 제보 요청과 답변
- 지도형 현장 제보 보기

웨더체크는 공식 재난 알림 서비스가 아닙니다. 위험 기상 상황에서는 기상청, 재난문자, 지방자치단체 안내 등 공식 정보를 우선 확인하세요.

## Keywords

날씨, 예보, 비, 강수, 기상청, 현장제보, 우산, 날씨비교, 지역날씨, 소나기
